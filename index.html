Element

An element is a part of a document that is defined by a start tag and an end tag

An element can have attributes that provide additional information about the element

Element are predefined

Element can be nested inside other elements

Element can be self-closing which is called self self-closing tag

Tags

A tag is a part of an element that defines the start and end of the element

A tag is enclosed in angle brackets

combination of tag opening and closing tag is called element

A tag can have attributes that provide additional information about the element

Attributes

an attribute is an additional information or feature to a tag

An attribute is defined in the start tag of an element

An attribute has a name and a value

An attribute is used to provide additional information about an element

An attribute is used to modify the behavior of an element
