Element

An element is a part of a document that is defined by a start tag and an end tag

An element can have attributes that provide additional information about the element

Element are predefined

Element can be nested inside other elements

Element can be self-closing which is called self self-closing tag

Tags

A tag is a part of an element that defines the start and end of the element

A tag is enclosed in angle brackets

combination of tag opening and closing tag is called element

A tag can have attributes that provide additional information about the element

Attributes

an attribute is an additional information or feature to a tag

An attribute is defined in the start tag of an element

An attribute has a name and a value

An attribute is used to provide additional information about an element

An attribute is used to modify the behavior of an element

Anchor tag

Anchor tag is used to create a link to another page or a section of the same page

Anchor tag is defined by the <a> tag

Anchor tag has an attribute called href which is used to specify the URL of the page or section to link to

Google

Anchor tag can also be used to link to a section of the same page by using the id attribute

how to link a page

< a href="https://www.example.com" target="_blank">Visit Example< /a>

target is an attribute of anchor tag which is used to specify where to open the linked document

_top is used to open the linked document in the same window or tab

_parent is used to open the linked document in the parent window or tab

_blank is used to open the linked document in a new window or tab

_self is used to open the linked document in the same window or tab

Form

form is used to collect the data and send it to other place

Form is defined by the <form> tag

Form has an attribute called action which is used to specify the URL to send the data to

form has 3 method

method is used to specify how to send the data

get method is used to send the data in the URL

post method is used to send the data in the body of the request

put method is used to update the data in the server
